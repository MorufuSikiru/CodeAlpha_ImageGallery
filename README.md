# 🖼️ Task 1 — Image Gallery

A responsive image gallery built with HTML, CSS, and JavaScript.

## 🔗 Live Demo

[View Live](https://morufusikiru.github.io/CodeAlpha_ImageGallery/)

## ✨ Features

- Responsive grid layout that works on all screen sizes
- Category filter buttons (Nature, Urban, Abstract, Portraits)
- Lightbox view with next/previous navigation
- Smooth hover effects and entrance animations
- Keyboard support (← → to navigate, Esc to close lightbox)
- Dark navy/electric blue theme

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript

## 📁 Project Structure

```
task1-image-gallery/
└── index.html
```

## 🚀 How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. No installation or dependencies required
